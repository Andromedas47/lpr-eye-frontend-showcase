# LPR Eye - Frontend Showcase 

โปรเจกต์ระบบตรวจจับป้ายทะเบียนรถแบบ Real-time (Senior Project) ที่เน้นการจัดการข้อมูลฝั่งหน้าบ้านด้วยเทคโนโลยีระดับสูง

### 🛠 Tech Stack ที่ใช้
- **Framework:** React 19 (Latest Version)
- **Language:** TypeScript สำหรับการทำ Type-safe
- **Build Tool:** Vite + Bun (เพื่อความเร็วในการ Development)
- **State & Routing:** TanStack Query & Router
- **Real-time:** Socket.IO สำหรับการรับข้อมูลป้ายทะเบียนทันที

### ✨ ความโดดเด่นของโปรเจกต์
- การจัดการ Form ที่ซับซ้อนด้วย **React Hook Form** และ **Zod**
- UI ที่สวยงามและ Responsive ด้วย **Tailwind CSS** และ **Shadcn/ui**
- การจัดการรันระบบผ่าน **Docker**
